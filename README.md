<h1>🎮 Cosmo Clash – Java Swing Quiz Gaming Application</h1>
<p>
Cosmo Clash is a desktop-based quiz gaming application built using <b>Java Swing</b>.  
It combines learning and fun through different game modes, a login system, and a leaderboard powered by CSV file storage.
</p>

<hr>

<h2>🚀 Features</h2>

<h3>🔑 Login & Signup System</h3>
<ul>
  <li>Secure login using <b>username & password</b> (stored in <code>users.csv</code>).</li>
  <li>Prevents duplicate accounts.</li>
</ul>

<h3>🏠 Home Screen with 4 Game Modes</h3>
<ul>
  <li>📝 <b>MCQ Quiz</b> – 10 shuffled questions, timed (15 seconds each).</li>
  <li>🤔 <b>Riddle Game</b> – Answer riddles with hints and limited tries.</li>
  <li>🗺️ <b>State-Capital Match</b> – Match states with their capitals.</li>
  <li>🏆 <b>Leaderboard</b> – View ranked scores with emojis/tags, stored in <code>quiz_leaderboard.csv</code>.</li>
</ul>

<h3>🎮 Game Mechanics</h3>
<ul>
  <li>Timer auto-skips unanswered questions.</li>
  <li>Scores calculated instantly at the end.</li>
  <li>Performance tags (e.g., <i>Master</i>, <i>Nice Try</i>).</li>
  <li>Dynamic leaderboard updated with each play session.</li>
</ul>

<hr>

<h2>📂 Project Structure</h2>
<p><b>📁 CosmoClash/</b></p>
<pre>
 ├── siyax_final.java              # Main application file
 ├── users.csv                     # Stores login credentials
 ├── space_geo_real_quiz.csv       # Quiz questions
 ├── guess_word_easy_moderate.csv  # Riddle questions
 ├── quiz_leaderboard.csv          # Leaderboard scores
 ├── screenshots/                  # App screenshots
</pre>

<hr>

<h2>⚡ How to Run</h2>
<ol>
  <li>Install <b>Java JDK 17+</b>.</li>
  <li>Open the project in <b>IntelliJ IDEA / Eclipse / VS Code</b>.</li>
  <li>Run <code>siyax_final.java</code>.</li>
  <li>Create an account → Choose a game mode → Play & track scores.</li>
</ol>
